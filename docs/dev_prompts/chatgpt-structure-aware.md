現在、添付ファイルのような状態に正規化されている文書ファイルがあります。これを、structure awareで、親子構造をつくりながら、チャンキングするpythonコードを書きたいです。cursorに投げられるプロンプトを作成してください。

#要件
-下記のHEADING_PATTERNSのようなものを利用して、改行や文書先頭の条件をつかってチャンクを分割してください。
HEADING_PATTERNS = ( 
    r"(?:"
    r"第[0-9一二三四五六七八九十百千〇]+(?:条|章|項|説|目)"
    r"|[0-9]+[.)．]"
    r"|[（(][0-9一二三四五六七八九十百千〇]+[)）]"
    r"|[•・\-]"
    r")"
    )

-文書の大半は正規化できていますが、OCRのズレで、一部改行が多かったり少なかったりする箇所があります。文書をよく読み込んで上手く判別できる仕組みを考えてください。
-外側の構造から再帰的に読み取って、2or3段階の親子構造を作るように設計してください。今回txtにしている文書以外は、構造化の形は近くても、HEADING_PATTERNの構造が一致しているとは限らないので、汎用性も考えてください。
-一つの.pyファイルで収まるように書いてください。
-テキストから変換する形でお願いします。現状は以下の簡単な分割なので、これらのchuncksize,overlapは一旦無視する形で実装してください。
def split_for_rag(*, text: str, chunk_size: int, chunk_overlap: int) -> list[str]:
    splitter = RecursiveCharacterTextSplitter(
        chunk_size=chunk_size,
        chunk_overlap=chunk_overlap,
        length_function=len,
    )
    return splitter.split_text(text)

-親チャンクと子チャンクの紐づけはmetadataに含まれるようにしてください。
